# Booked

A student-to-student textbook marketplace exchange platform built for a university project.

## Program Overview

Booked is a web-based C2C (consumer-to-consumer) platform that allows students to buy and sell second-hand textbooks. Once an order is placed, the seller's contact details are revealed to the buyer to arrange pickup.

## Tech Stack

- **Frontend:** HTML, CSS, BootStrap 5
- **Backend:** PHP
- **Database:** MySQL
- **Version Control:** Git & GitHub
- **Local Development:** phpMyAdmin
- **Hosting:** InfinityFree

## Features

- User registration and login
- Student textbook listings with title, author, ISBN, condition, and price
- Academic category filtering
- Listing expiry system (30 days)
- Order placement with seller contact detail reveal
- Report/flag system for suspicious listings
- Admin RBAC dashboard to manage users, listings, and reports
